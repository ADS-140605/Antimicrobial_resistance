Explainable AI — learning workspace

This folder is a sandbox for learning and experimenting with explainable AI (XAI) techniques.

Suggested contents:
- notebooks/: interactive experiments and explanations (SHAP, LIME, Captum)
- src/: implementation code for model explainers and visualization helpers
- experiments/: reproducible example runs and outputs

Starter actions:
1. Try a small notebook `notebooks/intro.ipynb` that loads a model and computes SHAP values.
2. Add exercises: feature attribution, counterfactuals, concept activation.
3. Ask me to scaffold a notebook, example explainer using Captum, or dataset loaders.

I can also add example notebooks or a `src/` module — tell me which to scaffold next.
